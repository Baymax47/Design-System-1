JavaScript 
const createToken = (name, value) => ({
  name,
  value,
  cssVar: `--ds-${name}`,
  toString() { return `var(${this.cssVar})`; }
});

const brand = createToken('color-brand-500', '#7c6ef7');
console.log(brand.toString()); // var(--ds-color-brand-500)
Inline code: npm run ds:build generates all output formats at once.

Confirm Action

This is a standard modal dialog. It traps focus within itself, closes on Escape or backdrop click, and restores focus to the trigger when dismissed.

Use Modal.open('id') and Modal.close('id') from JavaScript, or wire HTML declaratively with data-modal-open attributes.
